== Description ==

This plugin will allow your customer or visitors to delete products from woocommerce checkout page. 

== Features ==

* Whenever a person will reach to your checkout page, he will have option to delete items directly from checkout page. 




== Installation ==

1. Install the Woocommerce Product Remover From Checkout Page via the WordPress plugin directory, or by uploading the files to your web server (in the `/wp-content/plugins/` directory).
2. Activate the plugin through the 'Plugins' menu in WordPress and thats all.
